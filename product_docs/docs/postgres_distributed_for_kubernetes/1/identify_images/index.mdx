Gather the following information before [installing EDB Postgres Distributed for Kubernetes](/postgres_distributed_for_kubernetes/latest/installation_upgrade)
if you want to deploy a specific set of images and image versions: 

1. Identify your repository name and retrieve your user token as explained in [EDB private image registries](/postgres_distributed_for_kubernetes/latest/private_registries/).
1. [Identify the image names for your environment](/postgres_distributed_for_kubernetes/latest/identify_images/identify_image_name). 
1. [Assemble your deployment command](/postgres_distributed_for_kubernetes/latest/identify_images/assemble_command).
